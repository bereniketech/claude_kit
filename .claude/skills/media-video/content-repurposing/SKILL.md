# Content Repurposing

Turn one content piece into 8–12 distribution assets.

---

## 1. The Content Atom

Every piece of content has a **core atom** — the central insight, story, or finding. Identify it first:

```
Core atom of "[Your Content]":
- Main claim: [one sentence]
- Supporting evidence: [2–3 points]
- Practical takeaway: [what to do]
- Hook moment: [most surprising or shareable part]
```

All repurposed content starts from this atom.

---

## 2. Repurposing Map

### Source: Blog Post → outputs
```
→ Twitter/X thread (one tweet per section, hook in first tweet)
→ LinkedIn article (same structure, professional tone)
→ TikTok/Reel script (60-second summary of key insight)
→ Email newsletter section (200 words + link to full post)
→ YouTube video outline (sections become chapters)
→ Infographic (main points as visual list)
→ Quote graphics (pull 3 quotable lines as images)
```

### Source: YouTube video → outputs
```
→ Blog post (transcript cleaned up + added context)
→ TikTok/Reel clips (3–5 best 30-second moments)
→ Twitter thread (main points as numbered thread)
→ LinkedIn post (key insight + link to video)
→ Newsletter issue (summary + timestamps)
→ Podcast episode (audio only, minimal editing)
→ Shorts (single best moment, vertical crop)
```

### Source: Podcast episode → outputs
```
→ Transcript + blog post (with added visuals/examples)
→ Audiogram (30-second clip with waveform animation)
→ 5 quote graphics (for Instagram/LinkedIn)
→ Twitter thread (key takeaways)
→ YouTube upload (waveform or static image + full audio)
→ Newsletter recap (main points, timestamps, takeaways)
```

### Source: Newsletter issue → outputs
```
→ LinkedIn article
→ Blog post (expand each section)
→ Twitter thread (one tweet per section)
→ TikTok (read the intro as hook + 3 key points)
→ Instagram carousel (7 slides: title + 5 points + CTA)
```

## 3. Format Specifications

### Twitter/X Thread
```
Tweet 1: Hook (strongest claim or stat)
Tweets 2–N: One insight per tweet, numbered (2/ 3/ 4/)
Last tweet: Summary + link to source content
Rule: Each tweet must stand alone as a complete thought
```

### LinkedIn Post (text)
```
Line 1: Hook (no more than 1 sentence — gets cut after 2 lines)
Lines 2–5: 1 idea per line, white space between
CTA: Question or link in comments (not in body)
Length: 150–300 words optimal
```

### Instagram Carousel
```
Slide 1: Title + hook visual
Slide 2–6: One point per slide, short text, clear visual
Slide 7: Summary + CTA (save, share, follow)
Cover image: same visual strategy as thumbnail
```

### TikTok/Reel (vertical video)
```
0–3s: Hook — most interesting claim or result
3–50s: Teach the 3 key points
50–60s: CTA + tease next piece
Text overlay on screen for every key point (30% watch muted)
```

---

## 4. Repurposing Workflow

```
1. PRODUCE: Create the original (blog, video, podcast)
2. EXTRACT: Pull core atom + 5 best quotes/moments
3. ADAPT: Reformat each output for its native platform
4. SCHEDULE: Spread distribution over 5–7 days (not all at once)
5. MEASURE: Track which format/platform drove most engagement
6. DOUBLE DOWN: Next week, make more of what worked
```

**Time estimate per repurpose:**
- Twitter thread from blog: 20 min
- LinkedIn post from video: 15 min
- TikTok clip from video: 30 min (trim + captions)
- Newsletter section: 20 min
- Instagram carousel: 45 min (design-heavy)

---

## 5. Quality Rules

- **Adapt don't copy-paste**: Each platform has its own language. Rewrite, don't just paste.
- **Platform-native**: LinkedIn post ≠ tweet ≠ newsletter. Tone and length differ.
- **Spacing matters**: Twitter and LinkedIn both need white space — never paste a dense paragraph
- **Original value**: Each repurposed piece should add something (context, example, angle) not in the original
- **Frequency**: Don't repurpose everything. Repurpose content that performed well or covers evergreen topics.
